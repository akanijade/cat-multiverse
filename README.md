# 🐱 Cat Multiverse Explorer 

A fun incremental game where you explore cat universes, collect cats, and unlock hidden bosses.

---

## 🎮 Features

- Clicker system (earn cats)
- Upgrade shop (double / auto cats)
- Rarity system (common → legendary cats)
- Idle offline progression
- Save / load system (export code)
- Hidden boss universe
- Animated universe transitions
- Mobile-friendly UI

---

## 🧠 How to Play

1. Click to earn 🐱 cats
2. Switch universes
3. Buy upgrades in the shop
4. Unlock rare cats
5. Reach 100+ cats to unlock boss mode

---

## 💾 Save System

Click:
- Export Save → copy code
- Import Save → paste code
